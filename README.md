# PROJETOS AGES

Nesse repositório será possível encontrar todos os projetos em que pude participar durante meus anos dentro da **Agência Experimental de Software** *(AGES)*.

A *AGES* é uma matéria dentro do currículo de **Engenharia de Software** da *PUCRS*, onde é possível realizar softwares para pessoas ou empresas que desejam uma *POC* (Prova de conceito). A mesma será desenvolvida em 6 meses por uma equipe de alunos de diversos semestres e um professor responsável.
